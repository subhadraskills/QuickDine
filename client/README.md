# QuickDine 🍽️

QuickDine is a full-stack restaurant table booking application currently under development. The project allows users to explore restaurants, reserve tables, and manage bookings through a modern and responsive interface.

## 🚀 Tech Stack

### Frontend

* React
* TypeScript
* Vite
* Tailwind CSS
* React Router

### Backend 

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication

## ✨ Planned Features

* User Authentication
* Restaurant Listing
* Search & Filters
* Table Reservation
* Booking Management
* Restaurant Owner Dashboard
* Admin Dashboard
* Responsive Design

## 📦 Installation

```bash
npm install
```

## ▶️ Run Development Server

```bash
npm run dev
```

🚧 This project is currently under development. More features and backend integration will be added as development progresses.
